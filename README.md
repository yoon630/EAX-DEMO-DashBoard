# 🤖 AI Agent 예측형 생산최적화 시스템 (EAX-DEMO-DashBoard)

본 프로젝트는 **AI Agent 기반 예측형 생산 최적화 시스템**의 시각적 데모 대시보드입니다.  
AI Agent가 스스로 데이터를 수집하고, 이상 패턴을 감지하며, 생산 조건을 최적화하는 과정을  
**3단계 계층 구조**로 시각화했습니다.

---

## 🌐 **Demo Site**
🔗 [https://yoon630.github.io/EAX-DEMO-DashBoard/](https://yoon630.github.io/EAX-DEMO-DashBoard/)

---

## 🧩 **구성 개요**

| 계층 | 설명 |
|------|------|
| 🧠 **지식 계층 (Knowledge Layer)** | 센서 데이터 및 품질 지표를 기반으로 예측 분석 수행 |
| ⚙️ **지식 구축 계층 (Construction Layer)** | AI가 데이터 수집 → 패턴 인식 → 예측 생성 → 액션 결정 과정을 순차적으로 수행 |
| 🚀 **실행 계층 (Execution Layer)** | 예측 결과에 따라 생산 계획, 설비 제어, 에너지 사용 등 자율 실행 |

---

✨ 주요 기능

📊 실시간 지표 시각화 — 설비 고장 예측, 품질 이상 감지, 생산 효율 모니터링

⚙️ AI 추론 과정 애니메이션 — 데이터 수집부터 액션 결정까지 단계별 시각화

🚀 자율 실행 시뮬레이션 — AI가 판단한 액션을 실시간으로 갱신 및 출력

🧠 프로젝트 목적

본 대시보드는 AI 기반 제조 환경의 지능형 의사결정 프로세스를 시각적으로 설명하기 위해 제작되었습니다.
특히, 생산 공정의 디지털 트윈화와 자율 최적화를 구현하는 AI Agent Workflow의 개념을 쉽게 이해할 수 있도록 구성되어 있습니다.
